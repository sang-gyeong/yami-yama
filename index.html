야미야마 이용 가이드

          
        

        
    
          
  1. 문제 준비: 문제 목록(JSON 형식)을 입력창에 붙여넣습니다.
    2. 
          
  2. 형식 확인: JSON 형식 가이드에서 예시를 보고, 복사해서 바로 수정해 사용할 수 있습니다.
    3. 
          
  3. 모드 선택: 문제별 즉시 채점 또는 마지막에 한 번에 채점을 선택합니다.
    4. 
          
  4. 문제 풀이: 답을 입력하고 제출하면 정답/해설을 확인할 수 있습니다.
    5. 
          
  5. 반복 학습: 결과 화면에서 틀린 문제만 다시 풀어 복습할 수 있습니다.
    6. 
          
  6. 저장 세트 활용: 생성한 문제 세트는 홈 화면에 저장되며, 제목 변경/삭제 후 다시 불러올 수 있습니다.
    7. 
          
  7. 무료 서버 동기화: 문제 내역은 기본 서버에 자동 저장/복원됩니다.
    8.
